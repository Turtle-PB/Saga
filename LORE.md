# LORE

Baseline lore for the current in-repo build of **WSL Saga Multiverse**.

> Scope note: this file describes only lore that is directly supported by `README.md` and the current playable arc text in `index.html`. Anything beyond that is intentionally left undefined here.

## Core premise

The setting treats **Windows Subsystem for Linux (WSL)** as a metaphysical gateway between incompatible digital civilizations. A rupture called **the Breach** exposes five major conflict arcs across the multiverse.  
See: [STORY_ARCS.md](STORY_ARCS.md), [GLOSSARY.md](GLOSSARY.md)

## Canon confirmed in the current build

- The player moves through **five arcs**.
- Each arc has a distinct realm, enemy set, boss, and victory transition.
- The major themes are **faction conflict**, **broken timelines**, **ghost data**, and **control over causality**.

Evidence:
- `README.md:22-40`
- `index.html:119-206`

## Canon boundaries

- The current playable build does **not** name a fully characterized protagonist.
- The current playable build does **not** define a save-state canon.
- Several factions and entities are introduced through dialogue, but long-form history is only partially present in-repo.
